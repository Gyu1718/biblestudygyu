# 반응형 플로팅 연구 도크 패치

## 적용 방법

이 ZIP의 내용물을 `Gyu1718/biblestudygyu` 저장소 루트에 그대로 업로드하고 덮어씁니다.

교체되는 파일:

- `assets/app.js`

새로 추가되는 파일:

- `assets/js/research-dock.js`
- `assets/css/research-dock.css`

홈페이지·원어성경·원어사전·연구 문서에서 이미 불러오는 `assets/app.js`가 연구 도크를 자동으로 로드합니다. 각 HTML 파일을 일일이 수정할 필요가 없습니다.

## 주요 기능

### 공통 기능

- 성서 연구 서고 홈 이동
- 원어성경 이동
- 원어사전 이동
- 현재 문서 목차 또는 장 목록
- 현재 책 연구 홈 이동
- 이전 장·다음 장 이동
- 600px 이상 스크롤했을 때 맨 위로 이동
- 페이지 종류에 따라 필요 없는 버튼 자동 숨김

### 화면 설정

- 시스템·라이트·다크 테마 직접 선택
- 선택한 테마를 브라우저에 저장
- 글자 크기 90%·100%·110%·120%·130%
- 원어성경 두 열·한 열 보기
- 원어사전 호버 켜기·끄기
- 화면 설정 초기화

## 반응형 동작

- 데스크톱: 오른쪽 아래 반투명 캡슐 버튼과 세로 플로팅 패널
- 모바일 세로: 화면 아래에서 올라오는 하단 시트
- 모바일 가로: 화면 오른쪽에서 나오는 사이드 시트
- 모바일 입력창 사용 중에는 닫힌 도구 버튼을 자동으로 숨김
- iPhone 안전 영역(`safe-area-inset-bottom`) 반영

## 접근성

- 키보드 Tab 이동
- Enter·Space 버튼 작동
- Esc로 하위 화면 또는 도크 닫기
- 포커스 순환
- ARIA 레이블과 확장 상태 제공
- 모션 감소 설정 존중
- 인쇄할 때 도크 숨김

## 기존 테마 버튼과의 관계

기존 `app.js`가 생성하던 작은 테마 전환 버튼은 연구 도크의 `화면 설정`으로 통합됩니다. 기존 버튼 요소는 호환성을 위해 생성되지만 새 도크 스타일에서 화면에 표시되지 않습니다.

## 확인 방법

업로드한 뒤 브라우저 캐시를 무시하고 새로고침합니다.

- Windows: `Ctrl + Shift + R`
- macOS: `Cmd + Shift + R`

다음 화면에서 확인합니다.

1. 홈페이지에서 오른쪽 아래 `도구` 버튼
2. 로마서·느헤미야 등 장별 연구에서 목차와 이전·다음 장
3. 원어성경에서 장절 선택과 한 열·두 열 보기
4. 원어사전에서 사전 검색과 원어성경 이동
5. 모바일 세로·가로 화면의 패널 전환

## 데이터와 콘텐츠

이 패치는 성경·사전·주석 데이터와 `catalog.js`를 수정하지 않습니다. 따라서 현재 등록된 로마서와 기존 연구 목록을 유지합니다.
