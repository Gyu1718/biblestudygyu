# 장절 호버링 자동화 v2.0 + 주석 칩 A–Z 팔레트

이 패키지는 이전 `주석 칩 v1.1`을 포함한 통합 수정본입니다. **v1.0과 v1.1은 업로드하지 말고 이 ZIP만 적용**합니다.

## 적용 파일

- 교체: `assets/js/bible-reader.js`
- 교체: `assets/css/bible-reader.css`
- 추가: `assets/js/commentator-chips.js`
- 추가: `assets/css/commentator-chips.css`

압축을 푼 뒤 위 경로 구조 그대로 저장소 루트에 덮어씁니다.

## 이번 개선

1. `body[data-book]`을 최우선으로 읽어 현재 성경책을 고정합니다.
2. `특히 1:16–17`의 마지막 글자 `히`를 히브리서 약어로 오인하던 문제를 해결했습니다.
3. `1:16–17` 같은 같은 장 범위와 `1:16–15:13` 같은 장간 범위를 구분합니다.
4. 긴 범위는 호버에서 시작 2절·끝 2절, 클릭 패널에서 시작 3절·끝 3절을 요약 표시합니다.
5. `히 1:1–3`처럼 성경책이 명시된 참조는 명시된 책을 우선합니다.
6. 상세 연구에서는 `16–17절`처럼 장을 생략한 표현도 현재 장 문맥으로 인식합니다.
7. 직접 통제할 때는 `data-bible-ref="ROM.1.16-15.13"`을 사용할 수 있습니다.
8. 자동 연결을 막을 때는 상위 요소에 `data-no-scripture-link`를 붙입니다.
9. 주석 칩은 이미 주석 칩으로 지정된 `span.c` 등에만 A–Z 고정색을 적용합니다.

## 대표 판정

- 로마서 페이지의 `특히 1:16–17` → 로마서 1:16–17
- 로마서 페이지의 `1:16–15:13` → 로마서 1:16–15:13
- 로마서 페이지의 `히 1:1–3` → 히브리서 1:1–3
- 문맥 없는 페이지의 `1:16–17` → 자동 연결하지 않음

## 캐시

업로드 뒤 `Ctrl + Shift + R` 또는 `Cmd + Shift + R`로 강력 새로고침합니다.
