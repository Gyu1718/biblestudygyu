# 홈페이지 로마서 복원 및 카드 밀도 개선 패치

## 적용 방법

ZIP을 풀어 `Gyu1718/biblestudygyu` 저장소 루트에 그대로 덮어씁니다.

교체되는 파일:
- `catalog.js`

추가되는 파일:
- `assets/css/home-compact.css`

## 변경 내용

1. `신약 연구` 섹션에 `로마서 심층 연구` 버튼을 복원했습니다.
2. 기존 `원어 사전` 항목은 유지했습니다.
3. 홈페이지 카드의 높이와 여백을 줄였습니다.
4. 데스크톱에서는 연구 카드를 2열 서가로 배치합니다.
5. 큰 파란 CTA 버튼을 작은 텍스트 링크로 바꿨습니다.
6. 긴 메타 설명과 소개문은 홈페이지에서 두 줄까지만 보여 줍니다.
7. 모바일에서는 한 열로 돌아가며 터치 영역은 충분히 유지합니다.

## 주의

이 패치는 로마서 콘텐츠 파일을 다시 포함하지 않습니다. 저장소의 `nt/romans/index.html`과 관련 파일이 이미 존재한다는 전제입니다.
