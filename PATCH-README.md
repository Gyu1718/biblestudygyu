# 장절 호버링 자동화 v2.1 + 주석 칩 A–Z 흰색 글자

이 패키지는 이전 통합본 v2.0의 수정본입니다. 아직 업로드하지 않았다면 **이 ZIP만 적용**합니다.

## 적용 파일

- 교체: `assets/js/bible-reader.js`
- 교체: `assets/css/bible-reader.css`
- 추가 또는 교체: `assets/js/commentator-chips.js`
- 추가 또는 교체: `assets/css/commentator-chips.css`

압축을 푼 뒤 위 경로 구조 그대로 저장소 루트에 덮어씁니다.

## v2.1 변경점

1. 모든 주석 칩 내부 글자색을 `#ffffff`로 통일했습니다.
2. 라이트·다크·시스템 테마에서 검은 글자로 전환되지 않습니다.
3. 흰색 글자의 가독성을 위해 일부 밝았던 칩 배경을 같은 색상 계열의 더 어두운 색으로 조정했습니다.
4. 일반 본문, 제목, 원어, 절 번호, 파싱·품사 배지에는 적용되지 않습니다.
5. A–Z 26개 칩은 계속 서로 다른 고정색을 사용합니다.
6. `HB`, `KD` 같은 복수 글자 약어도 기존과 동일하게 고정색을 받습니다.

## 기존 장절 호버링 개선 유지

- `body[data-book]`을 최우선으로 읽습니다.
- `특히 1:16–17`의 `히`를 히브리서 약어로 오인하지 않습니다.
- `1:16–17`과 `1:16–15:13`을 구분합니다.
- 긴 범위는 시작·끝 본문을 요약 표시합니다.
- `data-bible-ref`와 `data-no-scripture-link`를 지원합니다.

## 캐시

업로드 뒤 `Ctrl + Shift + R` 또는 `Cmd + Shift + R`로 강력 새로고침합니다.
