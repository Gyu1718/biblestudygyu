# 시편 플로팅 버튼 좌표 핫픽스 v1.1

이미 시편 Git 패키지 v1을 업로드한 경우 이 핫픽스만 저장소 루트에 덮어씁니다.

교체 파일:

- `ot/psalms/psalms-suite.css`
- `ot/psalms/psalms-suite.js`

수정 사항:

- 화면 설정 연구 도크와 `편 찾기` 버튼의 우측 하단 겹침 제거
- 연구 도크 실제 위치 기준 자동 좌표 계산
- 모바일·화면 회전·창 크기 변경 대응
- 연구 도크가 열린 동안 `편 찾기` 자동 숨김

업로드 뒤 강력 새로고침을 실행합니다.

- Windows: `Ctrl + Shift + R`
- macOS: `Cmd + Shift + R`
