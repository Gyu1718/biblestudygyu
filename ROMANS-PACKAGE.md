# 로마서 GitHub 병합 패키지

이 패키지는 `Gyu1718/biblestudygyu` 저장소의 루트에 그대로 덮어쓰도록 구성했습니다.

## 추가되는 기능

- `nt/romans/index.html`: 로마서 서재 및 1–16장 진입 화면
- `nt/romans/study/ch01~16.html`: 개역개정 본문 중심의 절별 성경 연구 허브
- 1–9장 심층 연구에서 공통 라이트/다크 모드와 성경 장절 팝오버 사용
- 각 절에서 해당 장별 심층 연구 단락으로 이동
- `data/krv_rom.json`: 로마서 16장 본문 데이터
- `data/links/rom.json`: 절↔심층 연구 이동 지도
- `catalog.js`: 신약 연구에 로마서 등록
- `tools/build_links.py`: 복수 절 앵커의 `data-verses` 지원

## 병합

압축을 푼 뒤 저장소 루트에서 파일 구조를 유지한 채 복사합니다. 기존 `catalog.js`와 `tools/build_links.py`는 이 패키지 버전으로 교체됩니다. 원래 로마서 연구 내용은 문장이나 주해를 수정하지 않았고, 공통 기능 연결과 절 범위 메타데이터만 추가했습니다.

## 현재 범위

- 종합 연구: 완료
- 장별 심층 연구: 1–9장 완료
- 절별 성경 연구 허브: 1–16장 완료
- 원어 파싱·인터라이너: 자료가 없어 추가하지 않음
- 로마서 관주 데이터: 원자료가 패키지에 없어 추가하지 않음. 저장소의 `tools/build_xrefs.py`로 `Rom`을 생성하면 허브에 자동 표시됩니다.
