# 원어 사전 데이터 출처와 사용 원칙

## 주자료

「히브리어 스트롱 사전」과 「헬라어 스트롱 사전」을 자동 추출했습니다.
각 항목의 `rawKo`는 원문 보존 필드입니다. 원본 파일명과 쪽수는 내부 검증용 메타데이터로만 유지하며 공개 화면에는 책 제목만 표시합니다.
원문 전사는 자동 처리 결과이므로 연구·출판에 사용할 때 원자료와 대조해야 합니다.

공개 GitHub Pages 배포 전에는 한글 번역·편집본의 재배포 권한을 사용자가 확인해야 합니다.

## 보충 자료

STEP Bible Data의 다음 파일은 한글 자료에서 비어 있거나 손상된 표제어·음역·품사·간단한 영문 정의를 검증하는 용도로만 사용했습니다.

- TBESG — Translators Brief lexicon of Extended Strongs for Greek
- TBESH — Translators Brief lexicon of Extended Strongs for Hebrew

STEP Bible Data: CC BY 4.0, Tyndale House Cambridge / STEPBible.org.
프로젝트 내 보충 필드는 `supplement.source`에 출처를 표시합니다.

## Strong 번호

헬라어 G3203–G3302는 원 Strong 체계에서 사용되지 않는 번호입니다. 이 패키지는 해당 번호에 빈 가짜 항목을 생성하지 않습니다.
