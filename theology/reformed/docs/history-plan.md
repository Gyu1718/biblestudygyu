# 역사 파트 설계안

## 1. 목적

역사 파트는 종교개혁 이후 개혁파, 장로교, 신앙고백, 개혁파 정통주의, 청교도, 신칼빈주의, 한국 장로교 수용사와 더불어 근대 자유주의 신학, 변증법적 신학, 칼 바르트와 신정통주의의 흐름을 연구 색인으로 정리하기 위한 영역입니다.

이 파트는 전체 교회사 서술이 아니라, 개혁신학과 신정통주의의 주요 교리 주제, 학자, 문헌, 성경 본문을 역사적 맥락 안에서 연결하는 것을 목표로 합니다.

## 2. 범위

포함 범위는 종교개혁과 개혁파 전통의 형성, 장로교 전통, 도르트 총회, 웨스트민스터 총회, 개혁파 정통주의, 청교도, 신칼빈주의, 한국 장로교 수용사, 근대 자유주의 신학, 변증법적 신학, 칼 바르트와 신정통주의입니다.

제외 범위는 초대교회사 전체, 중세교회사 전체, 동방교회사 전체, 로마가톨릭 교회사 전체, 현대교회사 전체, 세계교회사 일반 연표입니다.

기준 문장은 다음과 같습니다.

> 개혁신학과 신정통주의를 이해하는 데 필요한 신학사만 다룬다.

## 3. 1차 핵심 항목

초기 개혁전통 데이터는 다음 항목으로 시작합니다.

| id | 제목 | 역할 |
|---|---|---|
| `history-index` | 개혁전통의 역사 | 전체 안내와 흐름도 |
| `reformation-to-reformed` | 종교개혁에서 개혁파 전통으로 | 주요 종교개혁 흐름 구분 |
| `reformed-and-presbyterian` | 개혁파와 장로교는 어떻게 다른가 | 개혁파, 장로교, 칼빈주의 용어 구분 |
| `synod-of-dort` | 도르트 총회와 개혁파 구원론 | 예정론, 은혜론, 항론파 논쟁 연결 |
| `westminster-assembly` | 웨스트민스터 총회와 장로교 신앙고백 | 신앙고백, 교회정치, 예배, 언약신학 연결 |
| `reformed-orthodoxy` | 개혁파 정통주의란 무엇인가 | 정통주의와 스콜라주의에 대한 오해 정리 |

신정통주의 확장 데이터는 다음 항목으로 시작합니다.

| id | 제목 | 역할 |
|---|---|---|
| `modern-liberal-theology-background` | 근대 자유주의 신학과 신정통주의의 배경 | 신정통주의가 반응한 역사적 배경 |
| `dialectical-theology` | 변증법적 신학의 등장 | 말씀 중심 신학의 출현 |
| `barth-and-neo-orthodoxy` | 칼 바르트와 신정통주의 | 바르트 신학의 중심 허브 |
| `reformed-orthodoxy-and-neo-orthodoxy` | 개혁파 정통주의와 신정통주의의 역사적 긴장 | 아카이브 전체 비교사의 허브 |

## 4. 페이지 공통 구조

각 역사 항목은 다음 순서로 정리합니다.

1. 한 문장 요약
2. 핵심 정의
3. 역사적 배경
4. 주요 인물
5. 주요 문헌
6. 핵심 신학 쟁점
7. 연결되는 교리 주제
8. 연결되는 책과 학자
9. 자주 생기는 오해
10. 연구 활용
11. 인용 위치

## 5. 데이터 파일

개혁전통 역사 데이터는 `/data/tradition-history.json`에서 관리합니다.

신정통주의 역사 데이터는 `/data/neo-orthodoxy-history.json`에서 관리합니다.

역사 입력 양식은 `/templates/history-entry.template.json`에서 관리합니다.

## 6. 데이터 연결 원칙

역사 항목은 독립 문서가 아니라 기존 데이터와 연결되어야 합니다.

- `relatedTopics`: `data/topics.json`의 주제 태그와 연결
- `relatedBooks`: `data/books.json`의 책 id와 연결
- `relatedAuthors`: `data/authors.json`의 학자 id와 연결
- `relatedPassages`: `data/passages.json`의 성경 본문 id 또는 표기와 연결
- `researchUses`: 설교, 논문, 브런치 글쓰기, 교리교육에서의 활용 방향 기록

## 7. 작성 원칙

- 공개 저장소에 저작권 있는 원문 전문을 올리지 않습니다.
- 역사 설명은 직접 요약하여 작성합니다.
- 긴 인용문은 넣지 않고, 필요하면 인용 위치만 남깁니다.
- 특정 전통을 단순화하거나 과도하게 변증하지 않습니다.
- 용어 구분을 우선합니다.
- 각 항목은 반드시 교리 주제와 연결합니다.
- 비교는 동일화가 아니라 차이를 정밀하게 보기 위한 장치로 사용합니다.

## 8. 확장 예정 항목

- `puritanism` — 청교도와 개혁파 경건
- `dutch-reformed` — 화란 개혁주의
- `neo-calvinism` — 신칼빈주의와 일반은총
- `old-princeton` — 프린스턴 신학과 성경론
- `korean-presbyterianism` — 한국 장로교와 개혁신학 수용
- `modern-reformed-theology` — 현대 개혁신학의 주요 흐름
- `barmen-declaration` — 바르멘 신학선언과 교회 고백
- `natural-theology-debate` — 바르트와 브루너의 자연신학 논쟁
- `barth-election-doctrine` — 바르트의 그리스도 중심 예정론
